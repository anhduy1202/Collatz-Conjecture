MAIN PROGRAM
Process command line args
Create message queue (mq_open)
Create producer thread (pthread_create)
for max_threads (come from cmd args)
  Create consumer thread (pthread_create)
Wait for producer thread to be done
while queue not empty
  sleep(1)
tell all consumer thread to stop (pthread_cancel)
wait for consumer thread to be done
for max_threads 
  join consumer thread (pthread_join)

CONSUMER THREAD

If var is even 
  divide by 2
  push to the back (mq_send)
Else if var = 1
  print out number and steps
Else
